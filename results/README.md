# Results

This directory contains cleaned result outputs from the xylanase thermostability workflow.

## Structure

| Folder | Content |
|---|---|
| `tables/` | Final CSV and Markdown result tables |
| `figures/` | Final figures grouped by analysis section |

The results are organised to follow the thesis workflow: sequence analysis, structure analysis, FoldX stability, docking, machine learning, molecular dynamics, and integrated candidate prioritisation.
