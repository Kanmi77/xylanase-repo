PDB-only FoldX and docking workflow manifest

Purpose:
This manifest records scripts and small result files used for the PDB-only FoldX stability and docking workflow.

Workflow scope:
- PDB-derived xylanase structures only
- FoldX RepairPDB / Stability analysis
- AutoDock Vina docking using xylobiose and xylotriose
- Final PDB-only ranking/candidate summaries

Important distinction:
This PDB-only workflow is separate from the MODELLER-derived WT/mutant MD validation workflow.
The PDB-only workflow used experimentally derived PDB structures or repaired PDB-derived structures.
It should not be described as the six-candidate WT/mutant MD workflow.

Expected important outputs:
- xylanase_structured_subset_with_foldx
- xylanase_structured_subset_with_foldx_norm
- foldx4_wt_stability
- top15_candidates
- top15_candidates_with_docking
- top15_final_ranked
