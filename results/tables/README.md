# Result tables

This folder contains the cleaned tables used for thesis reporting and GitHub release.

| Folder | Analysis layer |
|---|---|
| `01_sequence/` | Curated dataset, sequence features, thermal labels, and conserved-position summary |
| `02_structure/` | Structural records, PDB structural features, and TM-align summary |
| `03_foldx/` | FoldX wild-type stability and mutation-screening tables |
| `04_docking/` | Docking comparison tables |
| `05_ml/` | FoldX-output machine-learning result tables |
| `06_md/` | Molecular-dynamics summary tables |
| `07_integration/` | Integrated candidate ranking and prioritisation tables |
