# Supplementary Material

This folder contains the supplementary material associated with the thesis:

**In Silico Structural Thermostability Assessment of Bacterial and Fungal Thermostable Xylanases through Bioinformatics and Machine Learning Predictions**

## Included Material

### Main_Thesis_Result_Tables

This folder contains the main thesis result tables used for downstream analysis, interpretation, candidate prioritisation, and reporting.

### Source_Result_Tables

This folder contains supporting source tables used to generate the final result summaries. These include FoldX, docking, and integrated candidate-ranking tables.

## Reproducibility Note

The computational workflow used to generate and organise the analysis is available in the main repository under:

- `workflow/`
- `scripts/`
- `config/`

The workflow is implemented using Snakemake and includes curation, sequence analysis, structural feature extraction, FoldX stability screening, docking, machine learning, and integrated candidate ranking.
